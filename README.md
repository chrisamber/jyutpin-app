**WaaPou** is a free bilingual leadsheet platform for working Chinese-dialect musicians. Search any song from a library of 3M+ tracks, get automatic Jyutping romanization with tone colouring, and export a print-ready leadsheet — in seconds, on any device.

No account. No paywall. No ads. Always free.

---

## Features

- **Instant Jyutping annotation** — every Chinese character gets its Cantonese romanization, auto-generated by an NLP engine used by linguists
- **6-tone colour system** — T1 red → T6 purple, so singers can see pitch contour at a glance
- **Entering tone markers** — entering tones (入聲) are visually distinct, the #1 pronunciation pitfall for learners
- **3M+ song library** — powered by [LRCLIB](https://lrclib.net), a free, open, CORS-friendly lyrics API
- **Teleprompter mode** — full-screen autoscroll for live performance, works on any device including Android
- **Print-ready leadsheet** — exports a clean A4/Letter PDF with title, artist, tone legend, and Amber Audio attribution
- **PWA — install on any phone** — works offline after first load, no app store required
- **Danger zone pedagogy** — demo song ships with curated pronunciation pitfalls and study notes

## Getting Started

```bash
git clone https://github.com/chrisamber/waapou.git
cd waapou
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

```bash
npm run build   # production build → dist/
npm run lint    # ESLint
npm run preview # preview the production build locally
```

No API keys. No environment variables. Everything works out of the box.

## Background

WaaPou was built to serve a gap that no existing tool fills: **working Chinese-dialect musicians who need pronunciation guidance and chord charts in one place.**

- **OnSong** ($36–72/year) handles live performance beautifully — but has no CJK support, no romanization, and is iOS-only
- **Jianpu sites** (51jianpu, jianpu.cn) have Chinese content — but are stuck in 2005, mobile-broken, and show no chords or romanization

WaaPou is free, works on Android, generates romanization automatically, and is purpose-built for the musician who sings in Cantonese but learned to read in English.
